custom

Custom fonts can make your charts have richer display capabilities.

Center on WebView container loading font family, than Echarts call it, now, let's get it!

Steps​

  1. Download your need font families;
  2. Change it to includes base64 CSS file;
  3. Use props fontFamilies;
  4. Echarts options configuration;

Change file type​

1. change​

Click there to change page

First，you needs checked Base64 encode options，then click Convert.

Import​

Use props fontFamilies, import your font family files：

import TESTCSS from './TEST';
import TESTCSS2 from './TEST2';

<RNEChartsPro
  ...,
  fontFamilies={[
    { fontName: "TEST", fontFile: TESTCSS },
    { fontName: "TEST2", fontFile: TESTCSS2 },
  ]},
  ...,
  />

Configuration​

In your echarts options configuration；

  const pieOption = {
    textStyle: {
      fontFamily: "TEST",
      fontSize: 15,
    },
    ...
  }
